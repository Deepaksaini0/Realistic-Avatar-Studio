Photoreal talking avatars · powered by D-ID API
● API Mode
How this works: This tool calls the D-ID API on their servers — that's where the actual photoreal video generation happens. You need an API key (free trial available at studio.d-id.com). Each ~15-second clip costs roughly 1–2 D-ID credits (free trial includes 14 credits).

1 D-ID API Key

Get a key: sign up at studio.d-id.com → API → copy key.

Render deployment (recommended): set DID_API_KEY as an env var in your Render dashboard → leave the API key field above blank. The browser never touches the key.

Local Node: npm install then DID_API_KEY=xxx npm start → open http://localhost:3000. Proxy URL = /api/proxy.

Local PHP (XAMPP): drop realistic-avatar.html + proxy.php in htdocs/video-maker/, start Apache, open http://localhost/video-maker/realistic-avatar.html. Change Proxy URL above to ./proxy.php.
